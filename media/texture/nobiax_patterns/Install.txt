
Nobiax_Patterns textures package 1
__________________________________


For install this package :

1) copy the forler "packages" into your Sauerbraten/Cube2 installation folder

2) Add this line at the end of the "data/default_map_settings.cfg" file (or in your map .cfg file) :

     exec "packages/textures/nobiax_patterns/package.cfg"


For use it :

Ingame, creat a /newmap, press F2 and look at the end of the list.


Enjoy
